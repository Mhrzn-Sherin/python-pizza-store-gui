#  Python Pizza Store GUI

A Python-based desktop pizza ordering system with a graphical user interface.  
Users can browse pizzas, select items, and generate an order using a simple and interactive GUI built with Tkinter.


# Application Type
Desktop GUI application built using Python Tkinter.


# Key Features
- Interactive graphical user interface (GUI)
- Pizza menu with images
- Order selection and management
- Simple and user-friendly design
- Image handling using Pillow


# Tech Stack
- Python 3.x
- Tkinter (GUI)
- Pillow (PIL)

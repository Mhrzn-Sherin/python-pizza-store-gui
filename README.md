# python-pizza-store-gui
A Python-based online pizza ordering system with user interface and order management.

Application Type
This is a desktop GUI application built using Tkinter.

Key Highlights
- Interactive graphical interface
- Image handling using Pillow
- Simple and user-friendly design

How to Run

1. Clone the repository:
   git clone https://github.com/Mhrzn_Sherin/python_pizza_store_gui.git

2. Navigate to the project folder:
   cd python_pizza_store_gui

3. Install dependencies:
   pip install -r requirements.txt

4. Run the application:
   python main.py
